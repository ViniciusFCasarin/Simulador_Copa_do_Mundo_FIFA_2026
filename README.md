## Linguagens

### <> HTML
 > É uma linguagem de marcação que funciona como o esqueleto da página, separando a página em seções que serão preenchidas pelo JavaScript. As seções podem ser títulos, textos, tabelas, entre outros.

### \# CSS
 > É uma linguagem de estilo que trabalha com o design da página. Ele usa as marcações do HTML para definir separadamente o layout, as cores, os espaçamentos e outros elementos visuais de cada seção da página.

### JS JavaScript
 
 > É onde atua o sistema lógico e de decisão do código. Ele é responsável por acessar os dados nos arquivos JSON, receber entradas do usuário, calcular os pontos das tabelas de times, tomar decisões de desempate, atualizar a tela do HTML e definir o fluxo dos times nas tabelas e mata-mata.
 Usa a função fetch() para buscar informações nos arquivos JSON em tempo real, sem precisar atualizar a página.

### {} JSON
 
 > Os dados do projeto são armazenados em formato JSON. Os arquivos .json são usados para definir a estrutura de dados, possibilitando padrões de armazenamento e acesso de dados.
